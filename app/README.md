# Crop Yield Prediction System
An AI-powered API that suggests the best crop based on soil and weather conditions.

## 🚀 How to Run
1. Activate Virtual Environment: `.\venv_314\Scripts\activate`
2. Start Server: `python -m uvicorn app.main:app --reload`
3. Access API Docs: `http://127.0.0.1:8000/docs`

## 📊 Technical Stack
- **Model**: XGBoost Classifier (Week 5)
- **Framework**: FastAPI (Week 3)
- **Data**: Kaggle 2,200-row Dataset (Week 4)
